## What is Image to Alpha Pattern?

Image to Alpha Pattern is a fully client-side web application that converts any photograph or graphic into a **color-coded grid matrix** — the format used by crafters for cross-stitch, bead art, and pixel art projects.

You upload an image, configure the grid dimensions, and the app automatically:

1. Divides the image into a cols × rows matrix of cells
2. Averages the pixel colors within each cell
3. Assigns each unique color a letter symbol (A, B, C … Z, AA, AB …)
4. Renders an interactive, zoomable pattern preview
5. Lets you edit individual cells with a pencil tool
6. Exports the finished pattern in multiple formats

**No account required. No data leaves your browser.**

## How it works

<Steps>
  <Step title="Upload your image">
    Drag your image onto the drop zone, or click to open the file browser. JPG, PNG, and WEBP files are supported.
  </Step>
  <Step title="Configure the matrix">
    Set the number of columns and rows. Enable **Lock aspect ratio** to automatically calculate the opposing dimension, keeping the pattern proportional to the original image.
  </Step>
  <Step title="Apply color filters (optional)">
    Use the **Color similarity threshold** slider to merge colors that are too close to distinguish. Use the **Alpha threshold** to discard near-transparent edge pixels from images with transparency.
  </Step>
  <Step title="Generate the pattern">
    Click **Generate Pattern**. Pixel averaging runs in a Web Worker so the UI stays responsive. The result is a grid where each cell has an averaged color and a letter symbol.
  </Step>
  <Step title="Review and edit">
    Inspect the pattern in the interactive canvas. Switch to the **Pencil** tool to repaint individual cells. Undo and redo changes as needed.
  </Step>
  <Step title="Export">
    Download your pattern as a high-resolution PNG image, an Excel workbook, a CSV for Google Sheets, or a standalone HTML page with color.
  </Step>
</Steps>
